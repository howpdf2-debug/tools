Found this tool useful?

Is my data safe and private?

Yes, completely safe. All processing happens directly in your browser. Your files never leave your device and are not uploaded to any server.

What is the maximum file size?

Each PDF file can be up to 100MB. There is no limit on the number of files you can merge.

Do I need to create an account?

No account or signup is required. This tool is completely free and works instantly.

Can I merge password protected PDFs?

Currently, only non-password protected PDFs are supported.
